# Component Overrides (ZALF)

Place customized components here.

Recommended file header template:

```jsx
/**
 * ORIGINAL PATH: js/plugins/.../OriginalFile.jsx
 * CUSTOM PATH: themes/zalf/components/.../CustomFile.jsx
 * REASON: short description of the customization
 */
```

Rule: when changing an import in a base file, keep the original import commented.
